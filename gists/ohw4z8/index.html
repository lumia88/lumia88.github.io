编译安装 Chrome 下 Vuejs 开发工具 vue-devtool

1.克隆项目 git 到本地

git clone https://github.com/vuejs/vue-devtools

2.在vue-devtools目录下安装依赖包

cd vue-devtools && cnpm install

3.修改插件设置manifest.json

vim ./shells/chrome/manifest.json

找到persistent:false修改为persistent:true

4.执行编译

npm run build

5.导入插件到 Chrome

打开 Chrome > 更多工具 > 拓展程序，开启“开发者模式” ，点击“加载已解压程序”按钮，选择刚刚编译好的插件路径：vue-devtools/shells/chrome，完成。

6.运行项目，F12 调起开发者控制台，选择 vue，开始调试。
